# License CLI

Simple LICENSE.md generator.

## Installation

To install type the following in the terminal:

```bash
pip install license-cli
```

## Usage

Command to generate LICENSE.md file looks like:

```bash
license <license-name>
```

For example:

```bash
license mit
```
